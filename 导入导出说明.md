# 配置导入导出功能说明

## 功能概述

支持将直播间列表和过滤器配置导出为 JSON 文件，方便备份、分享和迁移。

## 使用方法

### 📤 导出配置

1. 点击页面上的 **"📤 导出配置"** 按钮
2. 浏览器会自动下载配置文件，文件名格式：`douyin_config_YYYY-MM-DD.json`
3. 配置文件包含：
   - 所有直播间信息（room_id, web_rid, title, owner）
   - 当前的过滤器规则

### 📥 导入配置

1. 点击页面上的 **"📥 导入配置"** 按钮
2. 选择之前导出的 JSON 配置文件
3. 确认导入信息
4. 系统会自动：
   - 导入所有直播间（已存在的会跳过）
   - 应用过滤器规则
   - 保存到本地配置文件

## 配置文件格式

```json
{
  "filter": "正则表达式过滤规则",
  "rooms": [
    {
      "room_id": "真实房间ID",
      "web_rid": "网页房间ID",
      "title": "直播间标题",
      "owner": "主播名称"
    }
  ]
}
```

## 使用场景

### 1. 备份配置
定期导出配置文件，防止数据丢失

### 2. 多设备同步
在不同电脑之间共享配置：
- 电脑A：导出配置
- 电脑B：导入配置

### 3. 分享配置
与朋友分享监控的直播间列表

### 4. 批量添加
手动编辑 JSON 文件，一次性添加多个直播间

## 注意事项

- ✅ 导入时已存在的直播间会被自动跳过
- ✅ 导入会覆盖当前的过滤器设置
- ✅ 导入后配置会自动保存到 `douyin_config.json`
- ⚠️ 直播间的运行状态不会被导出/导入
- ⚠️ 确保 JSON 文件格式正确，否则导入会失败
